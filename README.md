Tricentis Vehicle Insurance Application – Selenium Automation Project

Project Overview :-

This project is an end-to-end automation testing framework developed using Java and Selenium WebDriver for the Tricentis Vehicle Insurance Application.

The automation suite covers multiple insurance modules including:

(1) Automobile Insurance
(2) Motorcycle Insurance
(3) Truck Insurance
(4) Camper Insurance

The project validates user workflows such as vehicle data entry, insurant details, product data selection, price option verification, and quote submission.

Tech Stack :- 
1. Language: Java
2. Automation Tool: Selenium WebDriver
3. IDE: Eclipse
4. Testing Type: Functional UI Automation
5. Browser: Google Chrome
6. Utilities Used:
   a. Apache Commons IO
   b. Screenshot Handling
   c. JavaScript Executor

Features :-
a. End-to-End Form Automation
b. Dropdown Handling
c. Dynamic Date Selection
d. Premium Validation
e. Quote Submission
f. Screenshot Capture

Selenium Concepts Used :-
(1). XPath & ID Locators
(2). Select Class
(3). JavaScript Executor
(4). Waits
(5). Conditional Validation
(6). Screenshot Handling
